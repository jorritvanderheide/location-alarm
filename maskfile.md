# Location Alarm

## analyze

> Run static analysis

```sh
flutter analyze
```

## test

> Run all tests

```sh
flutter test
```

## fmt

> Format all files

```sh
nix fmt
```

## fmt-check

> Check formatting without modifying files

```sh
nix flake check
```

## build-apk

> Build release APK

```sh
flutter build apk --release
```
## clean

> Clean build artifacts

```sh
flutter clean
```

## deps

> Get dependencies

```sh
flutter pub get
```

## run

> Run the app in debug mode

```sh
flutter run
```
